## 得分计算

### 一、基础得分
基础得分由比赛的级别得分和奖项得分相加得出。级别得分和奖项得分分别通过预先设定的对应关系来确定，具体如下：

#### 1. 比赛级别得分
| 比赛级别 | 得分 |
| ---- | ---- |
| 国际级 | 10 |
| 国家级 | 8 |
| 省级 | 6 |
| 市级 | 4 |
| 校级 | 2 |
| 其他 | 1 |

#### 2. 奖项得分
| 奖项 | 得分 |
| ---- | ---- |
| 特等奖 | 5 |
| 一等奖 | 4 |
| 二等奖 | 3 |
| 三等奖 | 2 |
| 优秀奖 | 1 |
| 其他 | 1 |

### 二、特殊比赛加分
若比赛属于特殊比赛（即比赛名称以 * 开头），则在基础得分的基础上额外增加 3 分。
